Additionally, a new tool named Malus exposed copyright loopholes by using AI to perform clean room clones of open source projects, raising massive software licensing concerns. Vercel reported an internal breach where an attacker accessed unencrypted environment variables through a compromised third party AI tool called Context AI. Lovable, an AI coding platform, suffered a zero click data breach that exposed source code and database credentials for all projects created before November 2025. Furthermore, Palo Alto Networks reported a massive increase in attacks targeting Kubernetes clusters through stolen tokens. ## Embedded Systems
Engineering teams are rapidly adopting Embedded DevOps to eliminate manual firmware testing bottlenecks by integrating hardware in the loop testing and over the air updates directly into cloud style CI/CD pipelines. To boost embedded software efficiency, Arm launched the Cortex A320, its first 64 bit Armv9 CPU core designed to run complex AI models and embedded Linux simultaneously at the edge with drastically reduced power consumption. Nvidia also partnered with Nokia to run distributed AI inferencing workloads directly on mobile network infrastructure using SR Linux software, improving spectral efficiency and enabling AI models to execute seamlessly on edge devices.
